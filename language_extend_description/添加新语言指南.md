# 添加新语言的步骤

## 1. 创建翻译文件

`src/i18n/` 下新建 `xx-XX.ts`，复制 `zh-CN.ts` 的内容，翻译所有字符串。

```ts
// src/i18n/ja.ts
import { UIStrings } from './types';
export const ja: UIStrings = {
  selection: { analyze: '分析', translate: '翻訳', ... },
  ...
};
```

## 2. 注册语言

两个地方：

### `src/types/index.ts` — 加枚举值

```ts
export const LANGUAGE_OPTIONS = {
  'zh-CN': '中文',
  'en':    'English',
  'ja':    '日本語',   
  'ko':    '한국어',    // ← 新增示例
} as const;
```

### `src/i18n/index.ts` — 注册翻译

```ts
import { ja } from './ja';  // ← 导入

const maps: Record<LanguageCode, UIStrings> = {
  'zh-CN': zhCN,
  'en': en,
  'ja': ja,     // ← 注册
};
```

## 3. 完成

设置页下拉框自动出现新语言。选择后全界面即时切换，不需要改任何组件代码。
