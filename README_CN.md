# Language Learn Plugin (语言学习插件)

基于AI的Obsidian语言学习工具。选择文本 → 获取解释 → 建立结构化笔记及关系网络。

## 快速开始

1. **安装** — npm install 后 npm run build,将main.js,data.json,style.css,manifest.json复制到Obsidian插件目录
2. **配置** — 打开设置 → Language Learn，粘贴您的API密钥，选择模型
3. **选择文本** → 点击「分析」浮动按钮 → AI解释并保存笔记

## 功能特性

### 单词分析
选择单词或短语 → 点击 **分析** → AI生成结构化笔记，包含：
- 词性 + 含义
- 发音（IPA / 拼音 / 假名）
- 词形变化（词根）
- 引申含义及演变路径
- 上下文例句

### 悬停预览
悬停在任何已知单词上 → 弹出窗口显示含义、词源、例句。点击可打开完整笔记。

### 上下文重新解释
点击笔记上的 **重新解释** → AI将现有解释与当前上下文进行比较：
- 适用时 → 引用笔记内容，引导思考
- 不适用时 → 解释新含义，可选择更新笔记

### AI聊天面板（右侧边栏）
与AI助手自由对话。支持 **流式输出**、**对话历史**（自动保存到 `_chat_histories/`）和 **文件浏览**（读取目录、搜索文件）。

### 词汇高亮
所有笔记中的已知单词会自动高亮显示。点击高亮单词可查看笔记。

### 关系图谱

生成单词之间的同义词/反义词/同源词关系。

**工作原理：**

1. **标记系统** — 每个单词在 `data.json` 中有3个标记：同义词/反义词/同源词，值为0（未处理）、1（已匹配）或-1（未找到匹配）。只有标记为0的单词进入队列。

2. **逐词查询** — 对于每个未处理的单词（焦点词），AI评估所有其他单词（候选词）：
   ```
   System: "分析哪些单词与 {focus} 存在 {cognate} 关系…"
   User: 所有其他单词列表
   ```
   
3. **思维链** — AI逐步推理：识别词根 → 比较 → 判定。不确定时返回 `[]`。

4. **检查器模型** — 第二轮AI检查每个匹配，排除虚构词源、伪同源词或牵强的关联。被检查器拒绝的条目标记为 `check: false` 并跳过。

5. **三个并行队列** — 同义词、反义词、同源词通过 `Promise.all()` 同时运行。进度显示在右侧面板。

6. **Wiki链接** — 确认的匹配会将 `[[wiki链接]]` 及推理写入两个单词的笔记：
   ```markdown
   ## 同义词
   - [[unchanged]] — 两者均表示"未改变"，可互换
   ```

7. **幂等性** — 标记为1或-1的单词不会重新进入队列。点击 **重置** 可清除所有标记和链接，然后重新生成。

**性能说明：** 此功能每种关系类型每个单词调用一次AI。100个单词约需300次调用。建议使用 `deepseek-v4-flash`（经济）或 `deepseek-v4-pro`（精准）。

## 设置

| 分类 | 选项 |
|------|------|
| **服务商** | OpenAI, DeepSeek, Gemini |
| **模型** | 各服务商预设下拉菜单，或自定义 |
| **语言** | 母语 + 目标语言（下拉菜单） |
| **存储** | 单词/短语笔记的自定义文件夹 |

所有设置在 **保存并应用** 后立即生效 — 无需重启。

## 支持语言

界面：英语、中文（可通过 `src/i18n/` 扩展）

分析：支持AI模型支持的任何语言。插件自动检测文本语言并调整发音格式（英语用IPA，中文用拼音，日语用假名）。

## 文件结构

```
vault/
├── _word_notes/          # 单词分析笔记
├── _phrase_notes/        # 短语分析笔记
├── _chat_histories/      # 聊天对话历史
└── _relations/           # 关系图谱调试日志
```

## 添加新语言

1. 复制 `src/i18n/en.ts` → 翻译所有字符串
2. 在 `src/i18n/index.ts` 的映射中注册
3. 添加到 `src/types/index.ts` 中的 `LANGUAGE_OPTIONS`

## 命令

| 命令 | 操作 |
|------|------|
| 分析选中的文本 | 分析单词/短语或翻译句子 |
| 强制存储选中的文本 | 强制进行单词级分析 |
| 重新分析当前单词 | 重新分析光标下的单词 |
| 生成关系图谱 | 运行同义词/反义词/同源词分析 |
| 刷新索引 | 清除所有关系标记并重新扫描文件夹 |
